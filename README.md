# User Review Card

User card application developed using React, Typescript. 

The project demo can be seen here-> https://review-card.netlify.app/

### Setting Up and Running

- Clone the  [repository](https://github.com/shantanutomar/user-review-card.git).
- Run `npm install` in the root directory.
- Run `npm start`.
- Hit the URL - http://localhost:3000
